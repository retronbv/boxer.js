# boxer.js

### Create error boxes, success boxes and more with ease!
```
   ╔═════════════════════════╗
   ║                         ║
   ║  ✔  boxer.js Success   ║
   ║                         ║
   ║   It all worked!        ║
   ║                         ║
   ╚═════════════════════════╝
```

```js
const box = require("@retronbv/boxer.js");
console.log(box.successBox("It all worked!", "boxer.js"));
```
